## Run Locally

```bash
  git clone https://github.com/MoovendhanR/tic-toc-toe-react
```

Go to the project directory

```bash
  cd tic-toc-toe-react
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run start
```
